## Requirements

- nodejs >= 12.14.1
- yarn >= 1.21.1

## Running

```bash
# clone this repo
# Install packages
yarn install
# Start server locally
yarn start
```

Application will be opened automatically on [localhost:3000](http://localhost:3000).

## Build

```bash
# Build for production
yarn run buld
```
